# Daily Activity Tracker

## Overview

This is a small project created to test various technologies. The main idea is to build a basic website for tracking daily activities.

## Features

* Add and manage daily activities
* Simple and minimal user interface
* Basic data tracking functionality

## Technologies

This project is intended as a sandbox for experimenting with different technologies. Depending on the implementation, it may include:

* Frontend frameworks/libraries
* Backend services
* Databases
* APIs

## Purpose

The goal of this project is not to create a production-ready application, but to:

* Explore new tools and technologies
* Practice development workflows
* Experiment with different approaches to building web applications
